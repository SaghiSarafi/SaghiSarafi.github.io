NYC Tree Census Map

This interactive web application is developed as my final project for GEOG 585L. It visualizes New York City's Tree Census data, allowing users to explore the distribution, species, and health of trees throughout the city. The app is built using modern JavaScript libraries such as Leaflet.js and D3.js, and aims to support environmental research and urban planning efforts.
